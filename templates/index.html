gTt-HIV e-learning Platform Data Solutions API

This is the API that offers models that are intergrated with the gTt-HIV e-leaning platform

This API has 3 endpoints:
/recom which uses GET
/retrain_abandon which uses POST
/predict_abandon which uses GET

1)To use /recom two parameters are required:
user_id
course_id
e.g./recom?user_id=5&course_id=6
This returns a JSON

2) Accessing the /retrain_abandon endpoint with the POST method will trigger a retrain of the abandonment prediction model on the latest data from the postgreSQL DB

3) To use the /predict_abandon endpoint, one parameter is required:
user_id
This return a prediction of abandonment for that user.
